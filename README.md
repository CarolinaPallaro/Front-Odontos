## Descripción del Proyecto

Este proyecto incluye:
- **Hooks en React**: Utilización de los Hooks de React para manejar el estado y efectos en componentes funcionales.
- **Componentización**: Creación de componentes reutilizables para una estructura más modular y mantenible.
- **Rutas**: Implementación de navegación entre diferentes vistas usando React Router.
- **Manejo del DOM**: Manipulación y control del DOM a través de React.
- **JavaScript Dinámico**: Desarrollo de componentes interactivos y dinámicos con JavaScript.
